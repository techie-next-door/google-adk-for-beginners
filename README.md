# google-adk-for-beginners
## Steps to run the code

- Clone this repo --> git clone https://github.com/techie-next-door/google-adk-for-beginners.git    
- Switch to google-adk-for-beginners-branch-1
- Edit the .env file with your crendentials and remove the .example extension
- run main.py using uvicorn --> To run: uvicorn main:app --reload
- Play around with your own state and see how session and runner works!
